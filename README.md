Bash scripts for multiple nix OSs

Setup Debian updates:

wget https://raw.githubusercontent.com/grom07/bash_scripts/master/upgrade.sh
chmod +x upgrade.sh
sudo mv upgrade.sh /usr/local/bin/upgrade

Exexute: upgrade
